راهنما: پروفایل

توضیحات

ماژول پروفایل برای مدیریت فیلدهای سفارشی پروفایل کاربران است.

نصب/حذف

هیچ اقدام خاصی لازم نیست، مراحل نصب استاندارد را دنبال کنید - پوشه /profile را در دایرکتوری ../modules استخراج کنید. ماژول را از طریق مدیریت -> ماژول سیستم -> ماژولها نصب کنید.

دستورالعملهای دقیق در مورد نصب ماژولها در راهنمای عملیات XOOPS موجود است.

دستورالعملهای عملکرد

برای تنظیم این ماژول، باید:

  • تنظیمات برگزیده خود را برای ماژول پیکربندی کنید (به "تنظیمات برگزیده" مراجعه کنید) و در صورت تمایل بلوک پروفایل را نیز پیکربندی کنید (به "بلوکها" مراجعه کنید).
  • ویرایش دستههای موجود یا افزودن دستههای جدید.
  • ویرایش فیلدهای موجود یا افزودن فیلدهای جدید. در اینجا میتوانید مشخص کنید که کدام فیلدها در کدام دسته قابل مشاهده خواهند بود و آیا در هنگام ثبت نام کاربر قابل مشاهده خواهند بود یا خیر.
  • تعیین ترتیب مراحل ثبت نام.
  • و در نهایت، میتوانید مجوزهای مربوط به فیلدهای فردی را تنظیم کنید - کدام فیلدها قابل ویرایش هستند، کدام فیلدها قابل جستجو هستند.

اقدامات ضد هرزنامه

برای به حداقل رساندن ثبت نامهای هرزنامه، مراحل زیر را انجام دهید:

  • به ماژول Protector در قسمت مدیریت بروید، به تنظیمات برگزیده بروید و سپس در پایین، این گزینه را پیدا کنید: "Stop Forum Spam" دادههای POST را در برابر اسپمرهای ثبت شده در پایگاه داده www.stopforumspam.com بررسی میکند.
    آن را روی "Ban the IP (no limit)" تنظیم کنید.

  • در فایل /class/captcha/config.php، مطمئن شوید که حالت روی "text" تنظیم شده است:

    return $config = array(
    'disabled' => false, // غیرفعال کردن CAPTCHA
    'mode' => 'text', // حالت پیشفرض، میتوانید 'text'، 'image'، 'recaptcha' (نیاز به کلید API دارد) را انتخاب کنید
    'name' => 'xoopscaptcha', // نام کپچا
    'skipmember' => true, // رد کردن بررسی CAPTCHA برای اعضا
    'maxattempts' => 10, // حداکثر تلاش برای هر جلسه
    );

  • در ماژول Profile، به قسمت مدیریت بروید و در مرحله Basic واقع در:
    /modules/profile/admin/step.php?id=1
    گزینه "Save after stop" را روی "No" تنظیم کنید.


  • در تنظیمات برگزیده Profile، گزینه "Use Captcha after the second Registration step" را روی "Yes" تنظیم کنید (مقدار پیشفرض است).
