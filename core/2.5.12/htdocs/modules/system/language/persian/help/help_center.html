چگونه از XOOPS استفاده کنم؟

XOOPS یک پلتفرم برنامه کاربردی وب است که با زبان PHP برای پایگاه داده MySQL نوشته شده است. جهتگیری شیءگرای آن، این ابزار را به یک ابزار ایدهآل برای توسعه وبسایتهای انجمن کوچک یا بزرگ، پورتالهای درون شرکتی و سازمانی، وبلاگها و موارد بسیار دیگر تبدیل کرده است. برای استفاده مؤثر از XOOPS، باید ماژولهایی را نصب کنید که قابلیتهای مورد نظر شما را ارائه میدهند. ماژولهای مختلفی در مخزن ماژولها وجود دارد، از تقویمها، گالریهای عکس، اخبار، تابلوی اعلانات گرفته تا بهروزرسانیهای آب و هوا.

چگونه از مرکز راهنما استفاده کنم؟

مرکز راهنما برای ارائه اطلاعات اولیه مربوط به ویژگیهای فردی بخش مدیریت به شما طراحی شده است. هر زمان که میخواهید اطلاعات بیشتری در مورد صفحه خاصی که در آن هستید بدانید، فقط روی نماد راهنما در گوشه سمت راست کلیک کنید.

چگونه میتوانم در XOOPS محتوا ایجاد کنم؟

برای این کار به یک ماژول نیاز دارید. این میتواند یک ماژول خبری یا یک ماژول محتوا باشد.
